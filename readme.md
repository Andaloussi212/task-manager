# 🗂️ Task Manager (Java)

## 📌 Description

Ce projet est une application console développée en Java permettant de gérer une liste de tâches.

L’utilisateur peut :

* Ajouter une tâche
* Afficher toutes les tâches
* Supprimer une tâche par son identifiant

Ce projet a été réalisé dans le cadre de mon apprentissage en BUT Informatique (1ère année).

---

## ⚙️ Fonctionnalités

* ➕ Ajouter une tâche (id, titre, description)
* 📋 Afficher les tâches
* ❌ Supprimer une tâche
* 🚪 Quitter l’application

---

## 🛠️ Technologies utilisées

* Java
* Programmation orientée objet (POO)
* ArrayList (structures de données)
* Scanner (gestion des entrées utilisateur)

---

## 📂 Structure du projet

* `Tache.java` → représente une tâche
* `GestionnaireTaches.java` → gère la liste de tâches
* `Main.java` → interface console (menu utilisateur)

---

## ▶️ Lancer le projet

### Avec IntelliJ IDEA

* Ouvrir le projet
* Lancer la classe `Main`

### En ligne de commande

```bash
cd src/main/java
javac model/*.java
java model.Main
```
